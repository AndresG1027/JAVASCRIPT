Astronomía

Selecciona un ejercicio del menú para comenzar.

1. Clasificación de Presión Arterial (IF)

Ingresa la presión sistólica y diastólica para clasificarlas.

2. Registro de Temperaturas (FOR)

Ingresa las temperaturas de los pacientes separadas por comas (ej: 36.5, 38, 37.2).

3. Conteo de Pacientes con Fiebre (WHILE)

Haz clic para ingresar temperaturas una por una. El programa se detendrá al ingresar **0**.

4. Clasificación de TRIAGE (SWITCH)

Ingresa un código de Triage (1 a 4) para ver la categoría.

5. Mediciones de Saturación SpO2 (DO-WHILE)

Haz clic para ingresar mediciones de SpO2 una por una. El programa se detendrá al ingresar **'no'**.

1. Clasificación de Calidad del Aire (IF)

Ingresa el valor AQI (Índice de Calidad del Aire) para clasificarlo.

2. Niveles de Ruido Ambiental (FOR)

Ingresa varias mediciones de decibelios (dB) separadas por comas.

3. Focos de Calor (WHILE)

Haz clic para ingresar temperaturas una por una. El programa se detendrá al ingresar **0**.

4. Clasificación de Residuos (SWITCH)

Ingresa un código (1-4) para identificar el tipo de residuo.

5. Monitoreo Nivel del Río (DO-WHILE)

Haz clic para ingresar niveles del río uno por uno. El programa se detendrá al ingresar **'no'**.

1. Clasificación de Brillo Estelar (IF)

Ingresa la magnitud aparente de una estrella para clasificar su brillo.

2. Distancias de Planetas (FOR)

Ingresa varias distancias planetarias (en millones de km) separadas por comas.

3. Conteo de Cráteres Lunares (WHILE)

Haz clic para ingresar diámetros de cráteres uno por uno. El programa se detendrá al ingresar **0**.

4. Identificar Cuerpo Celeste (SWITCH)

Ingresa un código (1-5) para identificar el cuerpo celeste.

5. Registro de Niveles de Luz (DO-WHILE)

Haz clic para ingresar niveles de luz uno por uno. El programa se detendrá al ingresar **'no'**.
